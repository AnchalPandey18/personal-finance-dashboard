💰 Personal Finance Dashboard

A modern React.js Personal Finance Tracker that helps users manage their income and expenses with interactive charts and analytics.

---

🚀 Features

✔ Add Income & Expense
✔ Delete Transactions
✔ Local Storage Persistence
✔ Expense Categories
✔ Pie Chart Visualization
✔ Bar Chart Analytics
✔ Dark Mode / Light Mode Toggle
✔ Responsive UI

---

🛠 Tech Stack

- React.js
- Vite
- Tailwind CSS
- Chart.js
- JavaScript (ES6)

⚙ Installation

Clone the repository

git clone https://github.com/AnchalPandey18/personal-finance-dashboard.git

Go to project folder

cd personal-finance-dashboard

Install dependencies

npm install

Run development server

npm run dev

---

📊 Features Explained

1️⃣ Transaction Management

Users can add income or expense transactions with descriptions and categories.

2️⃣ Data Persistence

All transactions are stored in localStorage, so data remains even after refreshing the page.

3️⃣ Analytics

Charts are used to visualize financial data:

- Pie Chart → Income vs Expense
- Bar Chart → Financial analytics

4️⃣ Dark Mode

Users can toggle between Light Mode and Dark Mode for better UI experience.
